Các lệnh Terminal cơ bản Windowns/Ubuntu

1. Truy cập thư mục
	cd  + folder (cd tech/sub-folder)

2. Tạo một thư mục mới
	md  + tên folder
	mkdir  + tên folder (mkdir tech hello world: tạo 3 thư mục liền lúc)

3. Tạo một file mới
	type nul >  + tên file
	echo nul >  + tên file
	touch + tên file

4. Liệt kê các thư mục
	dir
	ls

5.1. Di chuyển các file
	move style.css ../../git-test (Từ folder User/gitX/1-commit Sang User/git-test)
	mv

5.2. Thay đổi tên file và thư mục
	move index.html about.html (di chuyển và đổi tên file)
	mv

6. Copy
	copy index.html about.html (copy và đổi tên file)
	cp

7.1 . Xóa file
	del (Xóa file del <tên file 1> <tên file 2> <tên file 3> ... <tên file n>)
	rm
7.2. Xóa thư mục	
	rd (remove directory: rd <tên thư mục 1> <tên thư mục 2> ... <tên thư mục n>)
	-rf

8. Mở file
	Thêm dấu " " vào file cần mở "terminal.txt"
	
		
9. Xóa màn hình đang hiển thị
	cls
	clear

10. Help
	help
	--help
    Exit CMD
	exit
